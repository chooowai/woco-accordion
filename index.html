A simple, user-friendly and lightweight jQuery accordion plugin with smooth animations powered by CSS3 transitions.
It was based on the Minimal Accordion/Drawer Plugin with jQuery and CSS3 developed by alxndrwcz.
This plugin is licensed under MIT and anyone can freely use it, copy it and modify it.

How to use it

  1. Include the Woco Accordion plugin’s CSS in the header.
    2. <link href="woco.accordion.css" rel="stylesheet">
  2. Include the Woco Accordion plugin’s JavaScript after you have loaded jQuery library.
    3. <script src="woco.accordion.js"><script>
  3. Create a DIV element which contains pairs of H1 element and DIV element.
    4. <div class="accordion">
      <h1>Section 1</h1> <div>
        Content 1<br>Content 1<br>Content 1<br>Content 1<br>
      </div> <h1>Section 2</h1> <div>
        Content 2<br>Content 2<br>Content 2<br>Content 2<br>
      </div> <h1>Section 3</h1> <div>
        Content 3<br>Content 3<br>Content 3<br>Content 3<br>
      </div>
    </div>
  4. Initialize the plugin.
    5. $(".accordion").accordion();
    OR
    Initialize the plugin with options. $(".accordion").accordion({
      //whether expanding mulitple section is allowed or not multiExpand: false, //slide animation speed slideSpeed: 500, //drop down icon dropDownIcon: "&#9660",
    });
